# 使用SDK切换各种语言的版本

## 安装SDK

```shell
curl -s "https://get.sdkman.io" | zsh
```

执行

```shell
source "/Users/dankai/.sdkman/bin/sdkman-init.sh"
```

添加到环境变量

## 安装指定语言的指定版本

首先先列出语言可安装的版本

```shell
sdk list java
```
